TravOps Chrome Extension

A Background Timer with countdown and clock alarms — keeps running even when you close the browser tab. Perfect for farm list reminders in Travian.

⏱ Countdown-Timer 🔔 Uhrzeiten-Alarme (4 Slots) 🔁 Loop-Modus 🔊 Alarm-Ton 🆓 Free
Bereit zum Download
Version 1.0 · Manifest V3 · Chrome / Chromium
⬇️  Extension herunterladen (.zip)

Installation guide

1
ZIP-Datei herunterladen und entpacken
Den Download-Button oben klicken. Die heruntergeladene travops-extension.zip entpacken — z.B. in deinen Downloads-Ordner.
2
Open Chrome and go to the extensions page
Enter chrome://extensions/ in the address bar and press Enter.
3
Entwicklermodus aktivieren
Oben rechts den Schalter „Entwicklermodus" einschalten — er wird blau.
4
„Entpackte Erweiterung laden" klicken
Den Button „Entpackte Erweiterung laden" oben links klicken, der nach dem Aktivieren des Entwicklermodus erscheint.
5
Select the unpacked folder
In the file dialog, select the folder where you extracted the ZIP (the folder containing manifest.json).
✓
Fertig!
Das Timer-Icon erscheint in der Chrome-Toolbar. Klicke darauf, um den Background Timer zu starten.
Note: The extension is installed as an "unpacked extension" since it is not available in the Chrome Web Store. Chrome may show a warning message on the next start — simply confirm with "Keep".
