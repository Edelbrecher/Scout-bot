🗺️ Map & World

🗺️

Karte

{{ badge('map') }}
🧩 Voraussetzungen
  • Eine Welt im Server-Setup ausgewählt
  • Optional: eigener Account verbunden, damit eigene Dörfer hervorgehoben werden
⭐ Vorteile
  • Vollständige Übersicht über alle Dörfer, Spieler und Allianzen der Welt
  • Filter nach Allianz, Spieler oder Koordinate, plus Distanzmessung
  • Teilen-Link für Kartenausschnitte — auch ohne Login einsehbar
📋 So nutzt du es
  1. /map öffnen, mit Maus oder Touch verschieben und zoomen
  2. In der Filterleiste nach Allianz, Spielername oder Koordinate suchen
  3. Auf ein Dorf klicken für Details (Spieler, Allianz, Bevölkerung)
  4. Sichtbaren Bereich über "🔗 Teilen" als Link kopieren und z. B. in Discord posten
{{ openlink('map', 'Karte') }}
⚔️

Angriffe

{{ badge('attacks') }} 🤖 benötigt Discord-Server
🧩 Voraussetzungen
  • Discord-Bot im Server eingeladen (Personal-Workspaces ohne Discord werden nicht unterstützt)
  • Kopie der "Eingehende Angriffe"-Tabelle aus dem Versammlungsplatz
⭐ Vorteile
  • Automatische Erkennung & Sammlung eingehender Angriffe für die ganze Allianz
  • Markierung als 👻 Fake / 💀 Hard / ⚠️ Low durch Leader/HC, sichtbar für alle
  • Discord-Alarm-Channel bei neuen Angriffen
📋 So nutzt du es
  1. Im Spiel: Versammlungsplatz → Tabelle "Truppenbewegungen" kopieren
  2. Auf /attacks einfügen — der Bot erkennt Angreifer, Ziel und Ankunftszeit automatisch
  3. Über "🚀 Automatisch einrichten" einen Discord-Alarm-Channel anlegen lassen
  4. Angriffe als Fake/Hard/Low markieren, damit alle Mitglieder den Ernst sofort erkennen
  5. Unter "Alliance Attacks" sehen alle Mitglieder die importierten Angriffe gesammelt
{{ openlink('attacks', 'Angriffe') }}
📡

Sector Monitor

{{ badge('sector_monitor') }}
🧩 Voraussetzungen
  • Player Pro Abo (Standard-Stufe für dieses Feature)
  • Definierter Kartenbereich (Mittelpunkt X/Y + Radius)
⭐ Vorteile
  • Automatische Überwachung eines Kartenausschnitts auf Veränderungen
  • Erkennt neue Dörfer, Abrisse und Bevölkerungssprünge
  • Ideal zur Beobachtung von Nachbarn oder Feindgebiet ohne manuelles Scouten
📋 So nutzt du es
  1. /map/sector-monitor öffnen
  2. Mittelpunkt-Koordinate (X/Y) und Radius des zu beobachtenden Bereichs eintragen und speichern
  3. Das System erstellt periodisch Snapshots des Bereichs
  4. Veränderungen (neue/abgerissene Dörfer, Bevölkerungsänderungen) erscheinen in der Liste
  5. Beobachtungsbereich jederzeit anpassen oder weitere Sektoren anlegen
{{ openlink('sector_monitor', 'Sector Monitor') }}
⏱️

Timer & Reminder

{{ badge('timer') }}
🧩 Voraussetzungen
  • Keine — optional: Travian-Profil-Link für Direktlinks zum Versammlungsplatz
⭐ Vorteile
  • Zentrale Übersicht laufender Countdown-Timer
  • Farmlist-Timer mit einstellbarem Erinnerungs-Intervall
  • Direktlink zum Versammlungsplatz spart Klicks
📋 So nutzt du es
  1. /timer öffnen
  2. Travian-Profil-Link einmalig hinterlegen (wird nur lokal im Browser gespeichert)
  3. Farmlist-Timer-Intervall einstellen (z. B. alle 30 Minuten erinnern)
  4. Seite im Browser geöffnet lassen — Erinnerung erfolgt per Browser-Benachrichtigung
{{ openlink('timer', 'Timer') }}
📜

Settle-Liste

{{ badge('settle_list') }}
🧩 Voraussetzungen
  • Geplante Koordinaten für neue Siedlungen
⭐ Vorteile
  • Verhindert doppelte Siedlungspläne innerhalb der Allianz beim Worldstart
  • Übersicht, wer wohin siedeln möchte
📋 So nutzt du es
  1. /settle-list öffnen
  2. "➕ Neuen Eintrag hinzufügen" — Koordinate, Spielername (optional) und Notiz eintragen
  3. Vor dem Siedeln die Liste prüfen, ob die Koordinate schon vergeben ist
  4. Nach dem Siedeln den Eintrag löschen oder als erledigt markieren
{{ openlink('settle_list', 'Settle-Liste') }}

Farming Intelligence

Verteidigung

{{ badge('defense') }}

⚙️ Operations

⚙️

Operations

{{ badge('operations') }}
🧩 Voraussetzungen
  • Alliance Pro Abo
  • Truppendaten der beteiligten Mitglieder importiert (über My Account)
⭐ Vorteile
  • Planung komplexer Allianz-Operationen mit mehreren Zielen und Wellen
  • Automatische Prüfung des Plans auf häufige Fehler
  • Benachrichtigung der zugeteilten Mitglieder direkt per Discord
📋 So nutzt du es
  1. /operations öffnen, "+ Erste Operation erstellen"
  2. Ziele hinzufügen — per Klick auf die Karte oder durch Eingabe der Koordinate
  3. Wellen mit Truppen, Mitglied und Sendezeit planen
  4. "Plan prüfen" nutzen, um Fehler wie zu wenig Truppen oder falsche Ankunftszeiten zu finden
  5. Über "📢 Benachrichtigen" die Wellen-Zuteilung an Discord senden
  6. Abgeschlossene Pläne ins Archiv verschieben — bei Bedarf später wiederherstellen
{{ openlink('operations', 'Operations') }}
🗡️

My Op Plan

{{ badge('attack_planner') }}
🧩 Voraussetzungen
  • Player Pro Abo
  • Mindestens eine Welle, die einem Leader in Operations zugewiesen wurde
⭐ Vorteile
  • Persönliche Übersicht aller zugewiesenen Wellen inkl. Countdown
  • Automatische Reisezeit-Berechnung zum jeweiligen Ziel
📋 So nutzt du es
  1. /my-operations öffnen
  2. Zugewiesene Wellen erscheinen automatisch, sobald ein Leader sie zuteilt
  3. Countdown "⏰ Nächste Welle — senden in" beachten und pünktlich abschicken
  4. Bei Verspätung die tatsächliche Sendezeit vermerken, damit der Leader nachjustieren kann
{{ openlink('attack_planner', 'My Op Plan') }}
